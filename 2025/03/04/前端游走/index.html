    [2]动态的“指针”this

    在 JavaScript 的面向对象编程中，必须通过 实例对象（也就是 this） 来调用它

    - javascript

    JS 的 this 是动态绑定的。它不看函数定义在哪里，只看函数被谁调用。

    • 核心口诀：谁调用它，this 就指向谁。

    四种应用(来自ai的回复)：

    ① 默认绑定（全局）
    ② 隐式绑定（对象调用）
    ③ 显式绑定（Hook 常客：call / apply / bind）
    ④ 箭头函数（固执派）

    箭头函数没有自己的 this，它会继承外层的 this。

    - java

    在 Java 里，this 永远指向当前实例对象。

    • 用途：主要用于区分“成员变量”和“局部变量”（比如构造函数里 this.name = name）。

    • 特点：编译时就确定了，基本不会变，非常老实。
